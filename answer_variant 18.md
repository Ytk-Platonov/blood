# Контрольный вопрос №18: Как в вашем коде реализована фильтрация и сортировка данных?

В моей реализации уровня доступа к данным фильтрация и сортировка 
реализованы в методе `findAll()` абстрактного класса `AbstractRepository`.

## Реализация метода findAll()

Метод принимает следующие параметры:
- `$where` - строка с WHERE условием (с плейсхолдерами)
- `$params` - массив параметров для подготовленного выражения
- `$orderBy` - строка с ORDER BY (проверяется по белому списку)
- `$limit` - ограничение количества записей
- `$offset` - смещение для пагинации

## Пример использования:
```php
// Получение активных доноров с группой крови II+, 
// отсортированных по дате регистрации
$donors = $donorRepo->findAll(
    "blood_type = :blood_type AND rh_factor = :rh_factor AND is_active = :active",
    [
        'blood_type' => 'II',
        'rh_factor' => 'positive',
        'active' => true
    ],
    'created_at DESC',
    10,  // limit
    0    // offset
);
```
